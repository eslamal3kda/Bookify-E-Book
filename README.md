# Bookify E-Book Library 

A modern e-book reader web application built with React and Vite.
Users can browse books, read them online, and enjoy a smooth UI.

## 🚀 Live Demo
https://bookify-e-library.netlify.app/


## ✨ Features
- 📱 Responsive design
- ⚡ Fast performance with Vite
- 🎨 Clean and modern UI
- 🌙 Light & Dark mode toggle
- 🔐 Simple authentication login system


## 🛠 Tech Stack

- React/Vite
- React-Router-Dom
- Tailwind
- Formik 
- Yup 
- React Icons
- React-Hot-Toast
- Zustand
- Tanstack/React-Query (Not Yet)
- Axios (Not Yet)

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/eslamal3kda/Bookify-E-Book.git
``` 
2. Go To Project Folder

``` bash 
cd bookify-e-book
``` 

3. Install Dependecies 
``` bash 
npm install
``` 

4. Run The Project 
``` bash 
npm run dev
```
